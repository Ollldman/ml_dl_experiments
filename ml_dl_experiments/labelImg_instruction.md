# Работа с LabelImg
Установим LabelImg. Простой способ — установить в терминале через pip:
```shell
pip3 install labelImg 
```
Запустим приложение. В терминале: 
```shell
labelImg 
```
Если при открытии приложения возникли проблемы, запустите его через виртуальное окружение, например через conda:
```shell
conda create --name labelimg_env python=3.8
conda activate labelimg_env
pip install labelImg 
```
После запуска LabelImg вы увидите простой интерфейс:
![LabelIMG](./labelimg1.png)

Чтобы потренироваться самостоятельно размечать данные, скачайте маленький датасет. 
Подготовка к разметке

1. На левой панели нажмите кнопку "Open Dir".
    
2. Выберите папку, где лежат скачанные сырые изображения для обучения(/transport_dataset/images/train/). В списке "File List" справа появятся картинки.
    
3. Выберите папку для сохранения разметки. На левой панели нажмите "Change Save Dir". Выберите папку, куда будут сохраняться .txt файлы (/transport_dataset/labels/train).
    
4. Выберите формат YOLO. Убедитесь, что на левой панели выбран формат "YOLO". Если там стоит "PascalVOC", нажмите на кнопку, чтобы переключить формат. 

Должно быть так:
![LabelIMG](./labelimg2.png)

Как размечать данные

1. Нажмите кнопку "Create RectBox" на левой панели.
    
2. Зажмите левую кнопку мыши и нарисуйте прямоугольник вокруг объекта.
    
3. Введите в него имя класса (например, cat) и нажмите "OK".
    
4. Нарисованная рамка появится в списке "Box Labels" справа.
    
5. Повторите для всех объектов на изображении.
    
6. Нажмите "Save" (или Ctrl+S). В папке для разметки (/transport_dataset/labels/train) появится файл с таким же именем, как у изображения, но с расширением .txt.
   
Получится примерно так: 
![LabelIMG](./labelimg3.png)

Если сделать разметку аналогично и выделить мотоциклы (класс — bike) и машины (класс — car), то в папке /transport_dataset/labels/train создастся файл в формате YOLO TXT: 1.txt — с таким содержимым:

```html

0 0.640767 0.515324 0.094278 0.204028
1 0.596879 0.475482 0.066320 0.108581
0 0.708062 0.447461 0.018205 0.050788
1 0.734395 0.444834 0.024057 0.042032
```
